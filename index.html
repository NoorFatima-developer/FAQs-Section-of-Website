What is the difference between == and === in JavaScript?

== performs type coercion, so it compares values after converting them to a common type. === is a strict equality operator, which compares both the value and the type without coercion.

What is the purpose of npm in Node.js

npm (Node Package Manager) is a tool for managing JavaScript libraries and dependencies in Node.js projects. It allows developers to install, update, and manage reusable code modules from a central registry.

What is an API, and why is it important in web development?

An API (Application Programming Interface) allows different software systems to communicate with each other. In web development, APIs enable the frontend (React) to interact with the backend (Node.js) by exchanging data using protocols like HTTP.

What is the purpose of the Virtual DOM in React

The Virtual DOM is a lightweight copy of the actual DOM (Document Object Model) that React uses to determine changes needed in the UI.
